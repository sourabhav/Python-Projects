# 📰 Fake Headline Generator

A fun Python project that generates random fake breaking news headlines.

---

## 📌 Features

* Random headline generation
* Unlimited headline combinations
* User can generate multiple headlines
* Exit anytime

---

## 🛠 Technologies Used

* Python 3
* Random Module

---

## 📚 Concepts Practiced

* Lists
* Loops
* Random Module
* String Formatting
* User Input

---

## ▶️ How to Run

```bash
python fake_headline_generator.py
```

---

## 📸 Sample Output

```
BREAKING NEWS:
Alien discovers secret pizza factory!

--------------------

Do you want another headline? (yes/no):
```
