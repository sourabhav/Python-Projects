# 📰 Fake News Generator

A simple Python project that generates random and humorous fake news headlines using predefined subjects, actions, and places. This project demonstrates basic Python concepts such as lists, loops, user input, and the `random` module.

## 🚀 Features
- Generates random fake news headlines
- Uses Python's `random.choice()`
- Allows multiple headline generation
- Beginner-friendly project

## 🛠️ Technologies Used
- Python 3
- Random module

## ▶️ How to Run
1. Clone the repository.
2. Run the program:
   ```bash
   python fake_news_generator.py
   ```
3. Enter **yes** to generate another headline or **no** to exit.

## 📌 Sample Output
```
BREAKING NEWS!!-- Yash Acting Movie Theatre

Do you want to generate another headline like this?? (yes/no): yes

BREAKING NEWS!!-- Modi Running America
```
