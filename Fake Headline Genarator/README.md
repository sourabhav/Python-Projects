# 📰 Fake News Generator

A simple Python project that generates random and humorous fake news headlines using the `random` module. This beginner-friendly project demonstrates the use of Python lists, loops, user input, and random selection to create entertaining headlines.

> **Note:** This project is created purely for learning and entertainment purposes. The generated headlines are fictional and should not be considered real news.

---

## 📌 Features

- Generates random fake news headlines.
- Uses predefined lists of subjects, actions, and places/things.
- Creates a different headline every time.
- Allows users to generate multiple headlines in one execution.
- Beginner-friendly Python project.

---

## 🛠️ Technologies Used

- Python 3.x
- Python `random` module

## ▶️ How It Works

1. The program stores different:
   - Subjects
   - Actions
   - Places/Things

2. It randomly selects one item from each list.

3. A fake headline is generated in the format:

```
BREAKING NEWS!!-- Subject Action Place/Thing
```

4. The user is asked whether they want to generate another headline.

5. The program continues until the user enters:

```
no
```
